## 二、后台访问方式

**后台登录地址：** `https://你的域名/admin/login.html`

> ⚠️ 后台地址不出现在任何前台页面上，访客无法直接进入。
> 即使知道 `/admin/` 路径，直接访问任何页面也会被自动跳转到登录页。

**默认账号密码：**
| 账号 | 密码 |
|------|------|
| admin | hjh@2026 |

> 🔐 **强烈建议上线前修改密码**，在 `admin/login.html` 第 86 行找到：
> ```js
> const ADMIN_USERS = [
>   { username: 'admin', password: 'hjh@2026', role: '超级管理员' }
> ];
> ```
> 将 `admin` 和 `hjh@2026` 改为你自己设定的账号密码。

## 三、部署到阿里云（推荐方案）

### 方案 A：阿里云 OSS 静态网站托管（最简单，无需服务器）

**费用：** 约 10-30元/月（OSS存储+流量）

**步骤：**
1. 登录 [阿里云控制台](https://oss.console.aliyun.com/)
2. 创建一个 Bucket（存储空间），区域选上海或华东
3. Bucket 设置 → 读写权限选 **公共读**
4. 上传所有网站文件（保持目录结构）
5. Bucket 设置 → 静态网站托管 → 开启
   - 默认首页填 `index.html`
   - 404 页面填 `index.html`
6. 记录 OSS 访问域名（如 `xxx.oss-cn-shanghai.aliyuncs.com`）

**绑定自定义域名（阿里云域名）：**
1. Bucket → 域名管理 → 绑定用户域名
2. 填入你的域名（如 `www.hjh-tech.com`）
3. 到阿里云域名控制台 → DNS解析 → 添加 CNAME 记录
   - 主机记录：`www`
   - 记录类型：`CNAME`
   - 记录值：OSS 提供的 CNAME 地址
4. 等待 DNS 生效（通常 5-30分钟）

**开启 HTTPS（强烈推荐）：**
- 申请阿里云免费 SSL 证书（每年可申请20个）
- 在 OSS 域名管理中上传证书并启用 HTTPS

---

### 方案 B：阿里云 ECS 云服务器 + Nginx

**费用：** 约 80-200元/月（2核4G入门型）

**步骤：**
1. 购买 ECS（推荐选上海、Ubuntu 22.04）
2. 安装 Nginx：
   ```bash
   apt update && apt install nginx -y
   ```
3. 上传网站文件到 `/var/www/hjh-website/`
4. 配置 Nginx：
   ```nginx
   server {
     listen 80;
     server_name www.你的域名.com;
     root /var/www/hjh-website;
     index index.html;

     # 禁止列目录（保护 admin 路径）
     autoindex off;

     location / {
       try_files $uri $uri/ /index.html;
     }
   }
   ```
5. 域名解析：添加 A 记录，指向 ECS 公网 IP

---

### 方案 C：阿里云 Web 应用托管 / 静态网站

适合不想维护服务器的用户，上传即可，操作与 OSS 类似。

## 四、上传文件到 OSS 的方法

### 方法1：使用阿里云 OSS 控制台（拖拽上传）
直接在浏览器控制台上传整个 `hjh-website` 文件夹。

### 方法2：使用 ossbrowser（桌面工具，推荐）
下载地址：https://help.aliyun.com/document_detail/61872.html
支持拖拽整个文件夹批量上传，操作最简便。

### 方法3：命令行工具 ossutil
```bash
ossutil cp -r ./hjh-website/ oss://你的bucket名称/ --acl public-read
```

## 五、域名备案说明

> 📌 **重要**：在中国大陆运营的网站，**必须先完成 ICP 备案**才能正常访问。

**备案流程（约15-30个工作日）：**
1. 登录阿里云备案系统：https://beian.aliyun.com/
2. 上传营业执照（公司名称：上海泓锦浩机械科技有限公司）
3. 法人身份证正反面
4. 等待工信部审核通过
5. 备案号填写到网站底部（代码中已预留位置）

> 如果域名已有备案，跳过此步骤直接部署。
